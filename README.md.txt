Opis Web Aplikacije

Kreirat ćemo web stranicu za kozmetički salon. Stranica će sadržavati opise pojedinih trermana kao i prosječno vrijeme trajanja tretmana. Realizirat ćemo rezervaciju termina preko baze podataka te prikaz slodobnih termina kao raspored gdje će se moći odabrati slobodni termin. Vlasnica salona će se moći prijaviti kao administrator te dobiti više podataka o rezerviranim terminima (ime i prezime, e-mail, broj telefona te vrstu tretmana).
